HTML Tags:

form-> 
    1. label
    2. input (single line input) [type = ""]
        1. text, numbers, tel, password
        2. url, email
        3. search
        4. checkbox, button, submit, radio [name attribute is must, should be same for same form to avoid multiple seletion]
        5. file, range, color
        6. date / week / month / time/ datetime-locale
    3. textarea (multi line input) [ rows="" | cols ="" maxlength=""]
    4. select & option (for dropdown)

semantic tag -> meaning to user and browser 
    -> header, nav, main, aside, section, article, footer
Layout of page:
header
nav
main content (aside and section)
footer

non semantic tag -> not understandable to user but understadable to browser
    -> div [blocking]
    -> span [non blocking]
